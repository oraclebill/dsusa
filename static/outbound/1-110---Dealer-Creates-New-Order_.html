110 - Dealer Creates New Order

    • Created on: Sun, Aug 30 (8 days ago)
    • Reported by: Bill Jones
    • Assigned to:
    • Milestone: Kickoff Demo
    • Status: New
    • Priority: High (2)
    • Component:
    • Model Update Required:

    Dealer will log into the system and be presented with a ‘dealer home page’ dashboard. This page offers the opportunity to create a new order. The dealer selects the ‘new order’ option and is taken to the order creation screen.
    The order creation screen has a number of sections. Some sections are mandatory, most are optional. The Dealer will complete all mandatory sections, possibly also completing some optional sections. Each section has it’s own validation. Any completed sections must be valid prior to order submission. The dealer can save the order at any point in the process. Completion of the order entry process requires attachment of a floorplan diagram. This can be done view upload or fax. The system will notify the dealer once the order is ready to be submitted for processing.
